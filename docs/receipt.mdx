---
title: "Receipt"
date: "2024-04-17"
---

The receipt object defines a transaction. A receipt consists of the ‘header’ information, and the ‘line item’ information. A valid receipt is where the header amount is the sum of the totals of all the line-items.

## Special types of line items

### Flight

E.g. Delta, British Airways

| Syntax         | Description |
| -------------- | ----------- |
| passenger_name | Title       |
| ticket_number  | Text        |
